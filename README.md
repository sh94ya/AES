Эта утилита на C# предназначена для шифрования и дешифрования текста и файлов с использованием алгоритма AES (Advanced Encryption Standard). Она разработана с использованием Windows Forms в Visual Studio.

## Описание

Данная утилита позволяет пользователю:
- Шифровать текстовые строки.
- Дешифровать зашифрованный текст.
- Шифровать и дешифровать файлы.
- Использовать алгоритм AES с различными ключами и режимами работы.

## Функционал

Шифрование и дешифрование текста: 
Пользователь вводит текст, и утилита выполняет шифрование или дешифрование.
Работа с файлами: Возможность выбора файла для шифрования или дешифрования. Защита конфиденциальной информации в файлах.
Настройки ключа: Пользователь может задать свой ключ и вектор инициализации для алгоритма AES.
Интуитивно понятный интерфейс: Удобный GUI, разработанный с использованием Windows Forms.

## Установка

1. Склонируйте репозиторий:
```bash
git clone https://github.com/sh94ya/AES.git
cd AES
```

2. Откройте проект в Visual Studio (AES.sln).

3. Скомпилируйте проект и запустите утилиту.

## Использование

Запустите утилиту.
Выберите режим работы: шифрование или дешифрование.
Введите текст или выберите файл.
Укажите ключ шифрования и вектор инициализации (по желанию).
Нажмите кнопку "Зашифровать" или "Расшифровать".
Сохраните полученный результат.

## Технические требования

    .NET Framework 4.7.2 или выше.
    Visual Studio 2019 или выше.
    Windows 10 или выше.
